# Albumes-que-escuche-2025
Contiene toda la base de datos de los 365 álbumes que escuche en 2025, escuché un álbum por día, mis amigos me recomendaron sus álbumes favoritos, por lo que todo análisis que se haga con referencia a el conjunto de datos que se utilizó, debe ser tomado como uso lúdico, ya que la muestra está completamente sesgada a mi círculo social.

En la carpeta de data sources se encuentran tanto el excel en el que se realizaron todas las modificaiones necesarias para poder tener limpio nuestro conjunto de datos, como diversas categorías que pueden ayudar en caso de que se quiera trabajar con los metadatos de nuestro conjunto. A su vez se encuentran todos los datos que hay en el excel en .csv por si se requiere hacer analisis de ellos directamente en alguna aplicación para dashboards.
